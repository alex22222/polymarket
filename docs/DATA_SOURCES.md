# BTC / ETH 历史数据源实测报告

生成时间: 2026-05-24
测试环境: 中国大陆网络

## 实测可用性

| 数据源 | 可用性 | 日线历史 | 分钟线历史 | 无需API Key | 限制 |
|--------|--------|----------|------------|-------------|------|
| **Binance data-api** | ✅ 完全可用 | 2017.08起 | 2020起(完整) | ✅ | 1000条/请求, 需分页 |
| **CryptoCompare** | ✅ 完全可用 | 2012起 | 有限 | ✅ | 免费版有频率限制 |
| Yahoo Finance | ❌ 超时 | - | - | - | 当前网络不可达 |
| Coinbase Pro | ❌ 超时 | - | - | - | 当前网络不可达 |
| Kraken | ❌ 失败 | - | - | - | 当前网络不可达 |

## 推荐方案

**首选 Binance data-api.binance.vision** — 数据最全、最稳定、无需认证
**补充 CryptoCompare** — 可获取2012-2017年的早期数据
